# junit-printer

pretty-prints a junit report

## usage

```
$ junit-printer junit.xml
+--------+--------+--------+--------+---------+-------+
|  NAME  | STATUS | PASSED | FAILED | SKIPPED | ERROR |
+--------+--------+--------+--------+---------+-------+
| pytest | passed | 8      | 0      | 0       | 0     |
+--------+--------+--------+--------+---------+-------+
```
